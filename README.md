# Club TRIAX Blog

Club TRIAX の公式ブログです。

**URL**: https://triax.github.io/blog/

## 記事の書き方

### 1. ファイルの作成

`_posts/` ディレクトリに以下の命名規則でファイルを作成します：

```
_posts/YYYY-MM-DD-タイトル.md
```

例：
```
_posts/2024-12-03-welcome-to-triax-blog.md
_posts/2024-12-15-practice-report.md
```

### 2. Front Matter（メタデータ）

記事の先頭に以下の形式で情報を記述します：

```yaml
---
title: 記事のタイトル
date: 2024-12-03 12:00:00 +0900
categories: [カテゴリ名]
tags: [タグ1, タグ2]
author: triax
---
```

#### 必須項目

| 項目 | 説明 | 例 |
|------|------|-----|
| `title` | 記事のタイトル | `練習レポート 12月` |
| `date` | 公開日時（+0900 は日本時間） | `2024-12-03 12:00:00 +0900` |
| `categories` | カテゴリ（1つ推奨） | `[練習報告]` |
| `tags` | タグ（複数可） | `[練習, オフェンス]` |

#### オプション項目

| 項目 | 説明 | 例 |
|------|------|-----|
| `author` | 著者名 | `triax` |
| `image` | サムネイル画像 | `path: /assets/img/posts/xxx.jpg` |
| `pin` | トップに固定 | `true` |
| `toc` | 目次の表示（デフォルト: true） | `false` |

### 3. 本文の書き方（Markdown）

```markdown
## 見出し2

### 見出し3

通常の文章はそのまま書きます。

**太字** や *斜体* が使えます。

- 箇条書き1
- 箇条書き2

1. 番号付きリスト
2. 番号付きリスト

> 引用文

[リンクテキスト](https://example.com)

![画像の説明](/assets/img/posts/image.jpg)
```

### 4. 画像の追加

1. 画像を `assets/img/posts/` に配置
2. 記事内で参照：

```markdown
![説明文](/blog/assets/img/posts/ファイル名.jpg)
```

### 5. カテゴリ・タグの推奨

#### カテゴリ（1記事1カテゴリ推奨）
- `お知らせ` - チームからの告知
- `練習報告` - 練習の様子
- `試合レポート` - 大会・試合結果
- `イベント` - チームイベント

#### タグ（複数可）
- `オフェンス`, `ディフェンス`, `スペシャル`
- `大会`, `練習試合`
- `新メンバー`, `チーム紹介`

### 6. 記事の例

```markdown
---
title: 12月の練習レポート
date: 2024-12-15 18:00:00 +0900
categories: [練習報告]
tags: [練習, オフェンス, ディフェンス]
author: triax
---

## 今週の練習内容

12月15日の練習では、オフェンスのフォーメーション確認を行いました。

### オフェンス

新しいプレーを3つ追加しました：

1. スラント
2. アウト
3. ポスト

### ディフェンス

ゾーンカバーの確認を重点的に行いました。

## 次回の予定

来週は試合形式の練習を予定しています。
```

## 公開の流れ

1. `_posts/` に記事ファイルを作成
2. GitHub にプッシュ
3. GitHub Actions が自動でビルド・デプロイ（約1分）
4. サイトに反映

## ローカルでのプレビュー（任意）

```bash
bundle install
bundle exec jekyll serve
# http://localhost:4000/blog/ で確認
```
